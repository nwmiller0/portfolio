Segway Site

This was one of the first websites I ever created. Even though this was for a class assignment I had a lot of fun building the site. Our objective with this site was to create a multipage website with properly structured HTML and CSS. We could choose from two different topics for the website content or create a site based on our own interests. I decided to choose from the segway option. Overall I was happy with how some of the elements on the site turned out. I was able to create a back to top button that worked properly. I also created and redesigned a logo for the top left of the site.
